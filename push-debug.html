🔔 푸시 알림 진단

아래 단계가 어디서 막히는지 보면 원인을 바로 알 수 있어요. 이 페이지는 다른 탭을 띄우지 말고 그대로 둔 채 실행하세요.

⬜브라우저 지원1
⬜알림 권한2
⬜서비스워커 등록3
⬜FCM 토큰 발급 + 저장4
⬜서버에서 즉시 발송5
⬜브라우저 직접 표시 테스트
위 주황 버튼을 누르면 FCM 없이 브라우저가 직접 알림을 띄웁니다. 이것조차 안 뜨면 100% Chrome/Windows 설정 문제예요.

핵심 판단: "표시 기능만 테스트"에서 알림이 안 뜨면 → 코드/FCM 문제 아님. 아래를 확인하세요:
① Windows 설정 > 시스템 > 알림 켜짐 + Google Chrome 항목 켜짐
② Windows 집중 지원/방해 금지 꺼짐
③ Chrome 설정 > 알림에서 "Chrome이 시스템 알림 사용" 관련 설정, 사이트가 차단 목록에 없는지
④ 이 탭을 최소화한 상태에서 다시 시도 (포그라운드 확인)
